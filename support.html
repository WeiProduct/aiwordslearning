使用指南

详细的功能介绍和使用方法

🚀 快速开始

  1. 下载并安装"AI背单词"应用
  2. 首次启动时选择您的目标考试类型（IELTS、TOEFL等）
  3. 设置每日学习目标和提醒时间
  4. 开始您的第一个学习会话

📚 选择词汇库

  • 国际考试：IELTS、TOEFL、GRE、GMAT、SAT
  • 国内考试：CET-4、CET-6、专业四八级、考研英语
  • 教材词汇：人教版、外研社等主流教材

您可以在设置中随时切换词汇库。

🎯 学习模式

  • 智能自适应：AI根据您的学习表现推荐单词
  • 顺序学习：按词汇表顺序逐个学习
  • 随机学习：随机顺序，增加学习趣味性
  • 间隔重复：基于遗忘曲线的科学复习

🎵 语音功能

使用内置的语音合成功能：

  • 点击单词即可听到发音
  • 可调节语速和音量
  • 支持英式和美式发音
  • 自动播放功能可在设置中开启

📊 学习统计

在"统计"页面查看您的学习数据：

  • 每日学习单词数量
  • 学习时长和连续天数
  • 掌握率和准确率
  • 学习曲线和趋势分析

故障排除

常见问题的解决方案

应用无法正常启动怎么办？

+

请尝试以下解决方案：

  • 确保您的设备运行iOS 18.0或更高版本
  • 重启应用或重启设备
  • 检查设备存储空间是否充足
  • 尝试重新安装应用
  • 如问题持续存在，请联系我们的技术支持

语音功能无法使用？

+

请检查以下设置：

  • 确保设备音量已开启且不在静音模式
  • 检查应用内的语音设置是否正确
  • 尝试使用耳机测试音频输出
  • 重启应用并重新尝试
  • 确保设备的语音服务正常工作

学习进度丢失了怎么办？

+

学习进度恢复方法：

  • 检查是否意外切换了词汇库
  • 确认是否在同一台设备上使用
  • 尝试重启应用刷新数据
  • 如果最近重装了应用，数据可能已被清除
  • 未来版本将支持iCloud备份功能

应用运行缓慢或卡顿？

+

提升应用性能的方法：

  • 关闭其他不必要的应用释放内存
  • 重启设备清理系统缓存
  • 确保应用已更新到最新版本
  • 检查设备存储空间是否充足
  • 在设置中开启性能优化选项

如何更改学习目标和设置？

+

设置修改步骤：

  • 打开应用主页，点击右下角"设置"选项卡
  • 在"学习目标"中调整每日单词数量
  • 在"学习偏好"中选择难度和模式
  • 在"语音设置"中调整语速和音量
  • 在"通知设置"中设置学习提醒
